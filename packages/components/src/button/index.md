## API

| 属性      | 说明             | 类型          | 默认值 |
| --------- | ---------------- | ------------- | ------ |
| style     | 设置图标样式     | CSSProperties | -      |
| className | 设置图标的 class | string        | -      |
| spin      | 是否有旋转动画   | boolean       | -      |
