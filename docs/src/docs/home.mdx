## 介绍

一个参照 Vant 开发的 React 移动端组件库。

## 特性

- 🚀 47 个高质量组件。
- 💪 使用 TypeScript 编写，提供完整的类型定义
- 📖 提供丰富的文档和组件示例
- 🍭 支持按需引入和 Tree Shaking
- 🍭 支持服务器端渲染
